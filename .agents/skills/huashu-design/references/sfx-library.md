## Guidelines

### Volume Recommendations (From `apple-gallery-showcase.md` Audio Dual-Track System)
- **SFX Main Track**: `1.0` (no attenuation)
- **BGM Background Track**: `0.4 ~ 0.5` (SFX cuts through clearly)
- **Multiple SFX Overlap**: Use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### FFmpeg Assembly Templates
```bash
# Aligning a single SFX to a timestamp:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection Decision Tree
1. **Tactile action present** (typing/clicking/swiping) → `keyboard/` or `ui/`
2. **Element entering/exiting** → `transition/`
3. **Container-level operations** (cards/modals) → `container/`
4. **Status feedback** (success/failure/notification) → `feedback/`
5. **Progress/time elapsed** → `progress/`
6. **Brand impact/key moments** → `impact/`
7. **AI magic/transformation** → `magic/`
8. **Command line/code demonstration** → `terminal/`

### Avoiding Overlapping Audio Clutter
- Maximum of `2 concurrent SFX` at any single timestamp
- Up to 3 concurrent SFX allowed when BGM is lowered below 0.3
- Mute all other SFX during brand impact (leave a 0.2s silence before the hit)

## Prompt Writing Principles (For Reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**Three Elements of a Good Prompt**:
1. **Physical description of the sound**: Which object, what action ("mechanical keyboard single key press")
2. **Texture/style constraints**: apple-style / ios-style / cinematic / retro
3. **Negatives to exclude**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
