## 4. PPT / Keynote Presentation

**Specifications**:
- Standard: 16:9 (1920×1080px)
- Widescreen: 16:10 (1920×1200px)

**Key Design Elements**:
- One core message per page (avoid cluttering)
- Clear typography hierarchy (Title 40pt+ / Body 24pt+ / Annotation 16pt+)
- Generous whitespace for better clarity during projection
- Image-to-text ratio at least 60:40
- Consistent visual system (colors, fonts, spacing)
